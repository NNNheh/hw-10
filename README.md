<h1>Домашнее задание к работе 2</h1>
Условие задачи

Программа предназначена для работы с прямоугольной трапецией.
Пользователю предлагается выбрать одно из действий:

Рассчитать площадь прямоугольной трапеции.

Вывести определение фигуры.

Нарисовать фигуру с помощью выбранного символа.

<h4>1. Алгоритм и блок-схема</h4>
Алгоритм

Начало

1.Вывести сообщение о выборе фигуры: Прямоугольная трапеция.

2.Вывести меню с тремя пунктами:

1 — Расчет площади.

2 — Определение фигуры.

3 — Рисование фигуры.

3.Считать выбор пользователя (choice).

4.В зависимости от выбора:

Если 1:

Запросить верхнее основание a, нижнее основание b и высоту h.

Вычислить площадь:

<img width="142" height="48" alt="image" src="https://github.com/user-attachments/assets/7d0b28c4-d701-4590-a943-4aa4a45e7bca" />


Вывести результат.

Если 2:

Вывести определение прямоугольной трапеции:
«Это трапеция, у которой один угол прямой».

Если 3:

Запросить символ для рисования ch.

Запросить высоту height, верхнее основание top_base, нижнее основание bottom_base.

Проверить, что нижнее основание длиннее верхнего.

Вычислить приращение длины строк:

Если другое значение: вывести сообщение об ошибке.
